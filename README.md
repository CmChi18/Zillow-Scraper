# Zillow-Scraper
This web scraper parses the data from a Carmel home listing on Zillow and prints it into a comma-separated format. In order for this program to work in another location, the page must be parsed differently. In the future, it could be updated to support every location without modification.

## Running the script
Enter the command 'python3 ZillowScraper.py' to run it and it will print out comma-seperated strings for each URL provided in urls.txt. This can easily be piped into a .csv file with 'python3 ZillowScaper.py -> output.csv'.
